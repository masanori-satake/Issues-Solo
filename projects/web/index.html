プロジェクト概要

Issues-Soloは、プライバシーを最優先に設計された、JIRA専用の閲覧履歴ツールです。 「どのタスクをいつ見たか」「コメントは編集途中か」をブラウザ内に安全に記録し、あなたの作業をサポートします。

特徴

JIRA特化

JIRAの課題キーとタイトルを自動抽出し、閲覧履歴としてリスト化します。

編集中状態の保存

コメントや説明文の入力かを検知し、保存。ブラウザを閉じても「書きかけ」の状態を把握できます。

完全ローカル実行

すべてのデータはブラウザ内の IndexedDB に保存されます。外部サーバーへの送信は一切行われません。

Material 3 UI

Google Material 3 (M3) デザインを採用。サイドパネルで作業を邪魔せず快適に操作できます。

インストール方法

  1. Chrome ウェブストアにアクセスします。
  2. 「Chromeに追加」ボタンをクリックします。
  3. 拡張機能の固定をオンにすると、サイドパネルからいつでもアクセスしやすくなります。
